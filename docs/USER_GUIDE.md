# 阿里智能业务员 Pro - 快速入门指南

## 📦 产品简介

阿里智能业务员 Pro 是一款专为外贸企业设计的 AI 自动化助手，帮助您：

- ✅ 自动回复阿里巴巴询盘，15 分钟内快速响应
- ✅ 智能客户跟进，促进复购成交
- ✅ 自动任务处理，节省人工操作时间
- ✅ 支持出口通和金品诚企两种店铺类型

---

## 🚀 一键启动

### Windows 用户

1. 双击 `start.bat` 文件
2. 等待系统自动安装依赖（首次启动需要 1-2 分钟）
3. 浏览器会自动打开 http://localhost:3000
4. 开始使用！

### Mac 用户

1. 双击 `start.sh` 文件（或右键选择"打开"）
2. 等待系统自动安装依赖（首次启动需要 1-2 分钟）
3. 浏览器会自动打开 http://localhost:3000
4. 开始使用！

> **提示**：首次启动需要安装 PM2 和项目依赖，后续启动只需 3-5 秒

### 功能说明

#### 1. 自动回复询盘

**用途**：当客户在阿里巴巴发送询盘时，系统自动分析并生成专业回复

**使用方法**：
1. 点击「一键启动」按钮
2. 系统会自动打开浏览器并登录阿里巴巴
3. 自动读取未读询盘并生成回复
4. 回复完成后会在日志中显示结果

**适用场景**：
- 有新询盘时需要快速响应
- 下班后/节假日自动回复
- 业务员忙碌时辅助处理

#### 2. 自动任务执行

**用途**：自动处理各种待办任务，包括客户跟进、数据同步等

**使用方法**：
1. 点击「启动自动任务」按钮
2. 系统会自动轮询任务队列
3. 发现任务后自动执行
4. 执行完成后更新任务状态

**适用场景**：
- 定期客户跟进提醒
- 数据自动同步
- 订单状态更新

#### 3. 店铺类型切换

**用途**：根据实际店铺类型调整策略和指标

**使用方法**：
- 点击顶部「出口通店铺」或「金品诚企店铺」按钮
- 系统会自动适配不同的响应时效和指标要求

## ⚙️ 系统配置

### 配置文件

在项目根目录创建 `.env` 文件，配置以下环境变量：

```bash
# 阿里巴巴后台地址
ALIBABA_MESSAGE_URL=https://message.alibaba.com/message/default.htm

# Cookie 存储路径（可选，默认为 data/cookies）
COOKIE_PATH=./data/cookies

# 浏览器配置（可选）
BROWSER_TIMEOUT=30000
```

### 浏览器要求

系统使用 Playwright 进行浏览器自动化，首次运行时会自动下载 Chromium 浏览器（约 150MB）

**网络要求**：
- 首次启动需要能访问 npm 和 GitHub
- 如果网络受限，可手动下载 Chromium

## 🔧 常见问题

### Q1: 启动后显示"PM2 未安装"
**解决**：系统会自动安装 PM2，请等待安装完成。如果安装失败，可手动运行：
```bash
npm install -g pm2
```

### Q2: 启动后浏览器没有自动打开
**解决**：
1. 检查是否允许了浏览器自动化权限
2. 手动在浏览器访问 http://localhost:3000
3. 查看日志是否有错误信息

### Q3: 显示"连接服务器失败"
**解决**：
1. 确认 PM2 服务是否正常启动
2. 运行 `pm2 status` 查看服务状态
3. 如果服务未启动，重启脚本

### Q4: 如何停止服务
**方法 1**（推荐）：关闭所有浏览器窗口，然后运行：
```bash
pm2 stop all
```

**方法 2**：直接关闭终端窗口（服务会继续在后台运行）

### Q5: 如何查看日志
```bash
# 实时查看日志
pm2 logs

# 查看 Web 服务日志
pm2 logs alibaba-web

# 清空日志
pm2 flush
```

### Q6: 如何重启服务
```bash
# 重启所有服务
pm2 restart all

# 重启单个服务
pm2 restart alibaba-web
pm2 restart alibaba-executor
```

## 🛡️ 注意事项

### 账号安全
1. **Cookie 存储**：登录 Cookie 存储在本地 `data/cookies/` 目录，不会上传到云端
2. **操作频率**：系统自动限制操作频率，避免触发阿里巴巴风控
3. **人工确认**：重要操作（如发送报价）建议人工确认后再执行

### 数据安全
1. **本地存储**：所有客户数据、任务数据都存储在本地 `data/` 目录
2. **定期备份**：建议定期备份 `data/` 目录
3. **隐私保护**：不要在公共电脑上使用本系统
