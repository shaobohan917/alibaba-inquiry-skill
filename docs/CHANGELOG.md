# 阿里智能业务员 Pro - 版本说明

**简化体验，专注核心功能** - 为外贸老板/业务员打造零门槛的 AI 自动化助手

### ✨ 新增内容

#### 1. 一键启动脚本
- **Windows**: `start.bat` - 双击即可启动，自动安装依赖
- **Mac/Linux**: `start.sh` - 双击或命令行执行
- 自动检测并安装 PM2
- 自动安装项目依赖
- 自动打开浏览器访问系统

#### 2. 简化版 Web 界面
- **新名称**: 阿里智能业务员 Pro（去掉"7 岗位 AI 智能体协同系统"等技术术语）
- **新标语**: "让外贸更简单 - 自动回复询盘 · 智能客户跟进"
- **简化功能**: 只保留两个核心按钮
  - 💬 自动回复询盘
  - ⚙️ 自动任务执行
- **去掉的内容**:
  - 7 岗位流程图
  - Agent 选择下拉框
  - 命令执行控制台
  - 技术架构展示
  - "总监管指挥中心"等术语

#### 3. 用户友好文档
- `docs/USER_GUIDE.md` - 快速入门指南
- 常见问题解答 (FAQ)
- 服务管理命令说明

### 🔧 优化内容

#### 1. PM2 配置优化
**文件**: `ecosystem.config.js`

**更改**:
```javascript
// 添加 autorestart: false - 由用户手动控制重启
autorestart: false  // 不自动重启，由用户手动控制
```

**原因**: 避免 PM2 自动重启导致用户困惑，改为手动控制

#### 2. Web 界面视觉优化
- 更大的功能卡片（300px 最小宽度）
- 更清晰的图标（4em 大小）
- 更友好的按钮样式
- 移动端响应式布局

#### 3. 状态指示优化
- 简化状态栏（3 个状态指标）
- 直观的绿灯/红灯指示器
- 实时任务概览（5 个统计指标）

### ❌ 删除内容

#### 1. 技术术语
- "Agent" → "助手"或"功能"

#### 2. 复杂功能

- 7 岗位网格展示

#### 3. 自动启动功能
- 取消开机自启配置
- 取消 PM2 自动重启
- 改为手动一键启动

### 📊 技术指标

#### 启动速度
- **旧版**: 需要手动执行多条命令（5-10 分钟）
- **新版**: 一键启动，3-5 秒（首次 1-2 分钟安装依赖）

#### 界面加载
- **旧版**: 完整技术架构图（约 800 行 HTML）
- **新版**: 简化功能卡片（约 400 行 HTML，减少 50%）

#### 操作步骤
- **旧版**: 需要理解 Agent、Executor、Supervisor 等概念
- **新版**: 只需点击「一键启动」按钮
