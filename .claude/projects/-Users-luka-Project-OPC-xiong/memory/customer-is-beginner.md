---
name: 客户是技术小白
description: 最终用户是技术小白，不懂任何技术，产品必须做到开箱即用、傻瓜式操作
type: feedback
---

**产品必须傻瓜式**：客户不懂技术，所以不能要求客户配置环境变量、安装 Node.js 依赖、运行命令行、配置 API Key 等任何技术操作。

**Why**：用户明确说明客户是技术小白，什么技术都不懂。当前基于 Claude Code Skill + 命令行的交付方式对这类用户完全不可用。

**How to apply**：所有产品设计必须遵循：1) 无需安装配置 2) 图形化界面 3) 一键启动 4) 内置 API Key 管理 5) 自动化程度最大化。任何需要技术知识的方案都要排除。
