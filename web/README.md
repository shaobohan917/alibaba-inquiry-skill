# 阿里国际站 7 岗位 AI 智能体 - 可视化执行系统

## 📖 功能说明

本系统提供 7 个 Agent 的可视化操作流程，支持在浏览器中直观查看：
- 7 岗位协同流程图
- 各 Agent 核心功能
- 数据流向图
- 核心框架层服务
- 实时命令执行控制台
- 出口通/金品诚企双版本切换

## 🚀 快速启动

### 方法一：直接使用 Python
```bash
cd web
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080
```

### 方法二：使用 Node.js (需安装 http-server)
```bash
cd web
npx http-server -p 8080
# 浏览器打开 http://localhost:8080
```

### 方法三：直接打开 HTML 文件
```bash
open web/index.html
# 或双击 web/index.html 文件
```

## 💡 使用指南

### 1. 切换店铺类型
- 点击顶部的 **出口通** 或 **金品诚企** 按钮
- 切换后各 Agent 的指标和策略会相应调整

### 2. 查看流程图
- 7 个 Agent 卡片展示在网格中
- 点击任意 Agent 卡片可选中该 Agent
- 数据流向图显示 Agent 之间的协作关系

### 3. 执行命令
1. 选择要执行的 Agent（如：物流报价员）
2. 选择命令（如：货代比价）
3. 输入参数（JSON 格式，可选）
4. 点击 **▶️ 执行** 按钮
5. 在控制台查看执行结果
