# 阿里智能业务员 Pro v2.0 前端

React + TypeScript + Vite + Tauri 桌面应用前端。

## 技术栈

- React 19 + TypeScript
- Vite
- Ant Design
- React Router
- Zustand
- Recharts
- Tauri v2

## 本地开发

```bash
npm install
npm run dev
```

## Tauri 桌面端

```bash
npm run tauri dev
npm run tauri build
```

## 路由

- `/login` 登录页
- `/console` 主控制台
- `/dashboard` 数据仪表盘
- `/sales` 业务员
- `/operation` 运营
- `/settings` 设置
